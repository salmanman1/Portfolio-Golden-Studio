# **App Name**: Golden Reels

## Core Features:

- Video Showcase: Display a curated collection of premium videos in an engaging grid layout.
- Video Playback: Enable users to stream and watch the showcased videos directly within the app.
- Category Display: Show list of page categories.
- AI-Powered Tagging: Automatically generate relevant tags for each video using a generative AI tool for easier searching and categorization.

## Style Guidelines:

- Primary color: Saturated gold (#D4AF37) to reflect luxury and premium quality.
- Background color: Dark, slightly desaturated black (#202020) to enhance contrast and visual depth.
- Accent color: Soft amber (#D47D37), to subtly draw attention to key interactive elements.
- Body and headline font: 'Playfair', a modern serif, to give a fashionable, high-end feel. If longer text is anticipated, use this for headlines and 'PT Sans' for body.
- Use minimalist, custom-designed play icons in a golden hue to match the site's aesthetic.
- Implement a dynamic, slightly tilted grid layout for video thumbnails to create a visually striking presentation.
- Incorporate subtle fade-in and transition animations to add polish and a sense of sophistication.
